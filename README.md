# SRPHP
基于swoft的http服务和xadmin的模块化开发框架。继承角色管理，管理员管理，订单管理，权限管理，系统设置，配置管理，管理员操作日志及模块化开发。
兼容普通用户及管理员用户的验证及模块权限区分。

配置SRPHP
1、将bean.php中DB1,DB2账号密码变为自己的 
2、将app\InstallDb中数据库文件数据入库 
3、请将首页地址指向xadmin目录 
4、配置好后输入php swoftcli.phar run -c http:start
